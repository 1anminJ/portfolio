동양미래대학교 2024 EXPO — 대상 수상

교내 최대 규모 작품전시회에서 1등상(대상)을 수상한 프로젝트입니다.

디지털 기기의 과도한 사용으로 인한 손 소근육 발달 지연 문제를 해결하기 위해, 컴퓨터 비전 AI 센서(LeapMotion)를 활용하여 손의 움직임을 실시간 분석하고 게임 기반으로 손 기능 개선과 발달을 돕는 재활 플랫폼입니다.

Spring Boot 3.3 Java 17 Spring Data JPA MySQL AWS EC2 AWS RDS Swagger React.js Unity LeapMotion SDK

💡 핵심 기능 및 기여

  • 게임 데이터 수집 API: Unity 게임에서 전송되는 손 동작 데이터를 수집·저장하는 REST API 설계 및 구현
  • 재활 진행 대시보드 API: 수집된 데이터를 바탕으로 재활 진행 상황을 시각화하기 위한 통계 API 구현
  • 회원 관리 시스템: 사용자별 게임 기록 및 재활 이력을 관리하는 인증·인가 시스템 구축
  • AWS 인프라: EC2 + RDS 기반 배포 환경 구성 및 운영

📋 프로젝트 주요 포인트

  • 기존 재활 기기의 고가·저접근성 문제를 게임 기반 저비용 솔루션으로 해결
  • 5개의 재활 게임 (달려라 고양이, 날아라 빗자루, 울려라 오케스트라, 뽑아라 마법채소, 그려라 마법진)
  • 데이터 분석 기반 재활 진행 시각화 대시보드 제공
  • 의료 기관 협력 및 B2C 시장 진출 가능성까지 제안서에 포함
