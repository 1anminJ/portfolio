Spring Boot Java WebFlux OpenAI API Whisper (STT) GPT React Vite

핵심 기능 및 기술적 도전

  • Non-blocking I/O — Spring WebFlux (WebClient)로 대용량 오디오 파일 스레드 차단 없이 처리
  • Whisper API 기반 STT — 다화자 회의 녹음을 높은 정확도의 텍스트로 변환
  • 프롬프트 엔지니어링 — JSON Mode 강제로 LLM 응답 구조화, 프론트엔드 파싱 로직 제거
  • 자동 요약 + Action Item 추출 — 긴 스크립트에서 3줄 요약과 실행 과제를 자동 추출
